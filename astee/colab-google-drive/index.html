Colab에서 Kaggle 데이터셋 다운받고 압축 해제하기
November 26, 2022

코랩에서 Kaggle dataset 다운받고 압축 해제하기

  1. 우선 캐글에 가입하여 프로필 탭의 Your Profile > Account > API 항목에서 Create New API Token을 누르면 API key가 포함된 kaggle.json 파일이 다운 받아진다.

  2. 코랩에서 새로운 노트북을 생성한다.

  3. 아래 명령어를 실행하여 kaggle을 설치한다.

    ! pip install -q kaggle

  4. 아래 코드를 실행하면 [파일 선택] 버튼이 나오는데, 버튼을 눌러 아까 다운받은 kaggle.json 파일을 업로드한다.

    from google.colab import files

files.upload()

  5. .kaggle 이라는 이름의 새 폴더를 생성하고, kaggle.json 파일을 저장한다.

    ! mkdir ~/.kaggle

! cp kaggle.json ~/.kaggle/

  6. 파일의 권한을 변경한다.

    ! chmod 600 ~/경로/.kaggle/kaggle.json

  7. 코랩과 구글 드라이브를 연동(마운트)한다.

    from google.colab import drive
 drive.mount('/content/drive')

  8. 아까 생성한 .kaggle 폴더가 위치한 곳으로 이동한다.

    cd /content/drive/MyDrive/경로/.kaggle

  9. 데이터셋을 다운받는다. (대부분 압축 파일로 받아지는 것 같다)

    !kaggle datasets download -d onurgunes1993/rico-dataset --force

    다른 사람과 공유된 드라이브에서 다운받으려 하다 보니 API key가 노출될 수 있다는 warning이 떠서 --force 옵션을 추가했다.

  10. 다운받은 파일을 압축 해제한다.

    !unzip rico-dataset.zip

    하… 이 과정에서 할 말이 좀 많은데 ㅠ 일단 압축 해제하는 데에 굉장히 오랜 시간이 걸린다. 파일의 크기마다 다르겠지만 나는 약 4시간 정도 걸렸다…
    사실 그 정도는 참을 만한데, 압축 해제 후 구글 드라이브에 업로드되는 데에도 굉장히 많은 시간이 걸린다. 다운부터 압축해제까지 약 11시간 가량 걸린 것 같다… 🥲

아무튼 이렇게 인고의 시간을 거치면 구글 드라이브에 압축해제된 파일을 확인할 수 있다!
