Image Segmentation [U-Net]
January 17, 2023

  • 이미지 세그멘테이션 Image Segmentation 한번에 끝내기

Semantic Segmentation

  • 분할(Segmentation)에서는 각 픽셀마다 하나의 클래스로 분류함
  • 이미지가 주어졌을 때, (높이 X 너비 X 1) 크기를 가지는 한 장의 분할 맵(Segmentation map) 을 생성한다.

목표

  • 각 픽셀마다 N개의 클래스에 대한 확률을 뱉어야 하므로, 정답은 (높이 X 너비 X N) 의 형태를 갖는다.
  • 각 픽셀마다 원-핫 인코딩(one-hot encoding) 형식을 따른다.

모델 학습

  • 분할(Segmentation) 작업을 위한 데이터셋 생성 비용은 매우 비싸다 -> 각 픽셀마다 어떤 클래스로 분류되는지 일일이 지정해야 하기 때문이다.)
  • 하지만 일반적인 CNN 분류 모델의 형식을 크게 바꾸지 않고 학습할 수 있다 -> 딥러닝 네트워크 구조가 어렵지 않다! 객체 탐지 분야의 네트워크 구조가 오히려 더 복잡한 편이다!

U-Net

  • U-Net 아키텍처는 U자형으로 생긴 네트워크이다.
  • 수축경로(Contracting path): 이미지에 존재하는 넓은 문맥(Context) 정보를 처리한다.
  • 확장경로(Expanding path): 정밀한 지역화(Precise loaclization)이 가능하도록 한다.

Convolution 연산의 종류

  • Down-sampling 목적: Strided Convolution - 너비와 높이가 감소(해상도 감소!)
  • Up-sampling 목적: Transposed Convolution - 너비와 높이가 증가(해상도 증가!)

U-Net Architecture

수축 경로(Contracting Path)

  • U-Net 아키텍처의 초반 부분
  • 2 * 2 Max pooling 사용 - 해상도(너비와 높이) 2배 감소
  • Conv 연산으로 채널 크기는 2배 증가
  • 일반적인 CNN 모델처럼 다음의 형식을 반복: Conv 연산들 -> ReLU -> Max pooling
  • 그래서 사실 U-Net의 인코더 부분에 이미 학습이 잘되어 있는 다른 분류 모델을 사용하는 경우도 많다

확장 경로(Expansive Path)

  • U-Net 아키텍처의 후반 부분
  • 2 * 2 Convolution (Up-convolution) 사용 - 해상도(너비와 높이)가 2배 증가
  • Conv 연산으로 채널 크기는 2배 감소
  • 수축 경로에서 처리된 feature 채널 레벨에서 연결(concatenation) map을 잘라내어 가져와 수행

U-Net의 Overlap-tile 전략

  • FCN 특성상 입력 이미지의 해상도에는 제한이 없다. 하지만 U-Net 구조상 출력 이미지의 해상도가 입력 이미지보다 작다!
  • 따라서 실제 필요한 영역보다 큰 범위의 패치를 삽입해서 결과를 뽑은 뒤, 그것을 출력 결과 패치로써 사용한다.
  • 이미지의 경계 부분은 Extrapolation을 사용한다 - 미러링(Mirroring) 활용

U-Net의 학습 방법: Objective Function

  • U-Net은 Segmentation을 위한 네트워크이므로, 픽셀 단위로 소프트맥스를 사용한다.
  • 학습을 위해 다음의 cross-entropy 손실을 사용한다. (true label만 고려하므로 일반 cross entropy 공식과 동일)
  • 세포(Cell)를 명확히 구분하기 위해 작은 분리 경계(Small separation border)를 학습한다.

U-Net의 데이터 증진 (Data Augmentation)

  • 의료 데이터에서는 학습 데이터의 수가 적은 경우가 많으므로, 데이터 증진이 필요하다.
  • 추가적으로 Elastic Deformation 방법을 사용할 수 있다. (아래 사진 참고)

3. 데이터 전처리

import numpy as np
from tensorflow import keras

class OxfordPets(keras.utils.Sequence):
  def __init__(self, batch_size, img_size, input_img_path, target_img_path):
    self.batch_size = batch_size
    self.img_size = img_size
    self.input_img_path = input_img_paths
    self.target_img_path = target_img_paths

  def __len__(self):
    return len(self.target_img_path) // self.batch_size

  def __getitem__(self, idx):
    # 가져올 때 batch 단위로 가져오기 때문에 다음의 과정이 필요함!
    i = idx * self.batch_size
    batch_input_img_path = self.input_img_path[i: i + self.batch_size]
    batch_target_img_path = self.target_img_path[i: i + self.batch_size]

    # input 이미지들을 batch 단위로 x에 넣음
    x = np.zeros((self.batch_size,) + self.img_size + (3,), dtype='float32')
    for j, path in enumerate(batch_input_img_path):
      img = load_img(path, target_size=self.img_size)
      x[j] = img

    y = np.zeros((self.batch_size,) + self.img_size + (1,), dtype="uint8")
    for j, path in enumerate(batch_target_img_path):
      img = load_img(path, target_size=self.img_size, color_mode="grayscale")
      y[j] = np.expand_dims(img, 2)
      y[j] -= 1  # 0, 1, 2, ... 로 label하기 위함

    return x, y

val_gen = OxfordPets(batch_size, img_size, val_input_img_paths, val_target_img_paths)

5. 모델 구성 - Down sampling & Up sampling

from tensorflow.keras import layers

def get_model(img_size, num_classes):
  inputs = keras.Input(shape=img_size + (3,))

  x = layers.Conv2D(32, 3, strides=2, padding='same')(inputs)
  x = layers.BatchNormalization()(x)
  x = layers.Activation('relu')(x)

  previous_block_activation = x

  # Down sampling
  for filters in [64, 128, 256]:
    x = layers.Activation('relu')(x)
    x = layers.SeparableConv2D(filters, 3, padding='same')(x)
    x = layers.BatchNormalization()(x)

    x = layers.Activation('relu')(x)
    x = layers.SeparableConv2D(filters, 3, padding='same')(x)
    x = layers.BatchNormalization()(x)

    x = layers.MaxPooling2D(3, strides=2, padding='same')(x)

    residual = layers.Conv2D(filters, 1, strides=2, padding='same')(previous_block_activation)
    x = layers.add([x, residual])
    previous_block_activation = x

  # Up sampling
  for filters in [256, 128, 64, 32]:
    x = layers.Activation('relu')(x)
    x = layers.Conv2DTranspose(filters, 3, padding='same')(x)
    x = layers.BatchNormalization()(x)

    x = layers.Activation('relu')(x)
    x = layers.Conv2DTranspose(filters, 3, padding='same')(x)
    x = layers.BatchNormalization()(x)

    x = layers.UpSampling2D(2)(x)

    residual = layers.UpSampling2D(2)(previous_block_activation)
    residual = layers.Conv2D(filters, 1, padding='same')(residual)
    x = layers.add([x, residual])
    previous_block_activation = x

  outputs = layers.Conv2D(num_classes, 3, activation='softmax', padding='same')(x)

  model = keras.Model(inputs, outputs)
  return model

model = get_model(img_size, num_classes)
model.summary()
